# Smart Hoop

PWA-приложение для тренировки бросков в баскетбол. Устанавливается на телефон как обычное приложение и работает прямо в браузере — без App Store и Google Play.

## О проекте

Smart Hoop помогает отслеживать тренировочные сессии: считает попадания и промахи, ведёт статистику и серии, а в будущем — автоматически определяет результат броска через камеру и AI-модель.

Разработка идёт поэтапно: сначала весь pipeline в manual-режиме, затем подключается AI-режим (камера и ONNX-модель).

## Технологии

- **Vue 3** + **Vite** — фронтенд
- **JavaScript** (без TypeScript)
- **PWA** — manifest, service worker, установка на домашний экран
- **Vue Router** — навигация между экранами

Планируется:

- ONNX Runtime Web — детекция мяча и кольца
- IndexedDB — постоянное хранение сессий
- MediaRecorder API — запись видео тренировок
- Canvas + Web Audio API — overlay и звуки

## Запуск

```bash
npm install
npm run dev
```

Сборка для production:

```bash
npm run build
npm run preview
```

Приложение деплоится на GitHub Pages по адресу `/smart-hoop/`.

| Экран | Описание |
|-------|----------|
| **Главная** | Стартовая страница с навигацией |
| **Сессия** | Активная тренировка с HUD и управлением |

## Текущие возможности

### Этап 1 — PWA scaffold ✅

- Мобильный интерфейс с навигацией
- Установка как PWA (manifest, service worker, иконки)
- Экраны: Главная, Сессия, Статистика, Настройки

### Этап 2 — Manual-сессия ✅

- **Старт / Пауза / Завершить** — управление тренировкой
- **Попадание / Промах** — ручной ввод бросков в manual-режиме (без камеры и AI)
- **HUD в реальном времени:**
  - попадания, промахи, всего бросков
  - процент попаданий
  - таймер сессии
  - текущая серия и лучшая серия
- Завершённые сессии сохраняются в памяти приложения
- Экран статистики показывает список завершённых сессий

## Планируемые возможности

- 📷 Камера телефона (задняя камера)
- 🎯 AI-детекция мяча и кольца (ONNX)
- 📐 Ручная калибровка кольца
- 🤖 Автоматическое определение попаданий и промахов
- 🔊 Звуковые уведомления (попадание, промах, предупреждения)
- ⚠️ Предупреждения (камера закрыта, кольцо не видно)
- 💾 Сохранение истории в IndexedDB (персистентно после перезагрузки)
- 🎥 Запись видео тренировки
- 📊 Детальная статистика и экспорт (JSON, CSV)
- ⚙️ Настройки (режим сессии: manual / ai, звук, чувствительность)
